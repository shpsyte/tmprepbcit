The CSS Technique of the Week!!!

The techique of this week is SELECTORS

What Are Selectors?

Selectors as patterns that match against elements in a tree. Most of the selectors from the Level 4 specification are pseudo-classes. Selectors have been with us since the beginning of CSS, but now they are at the fourth level and have gotten a lot of cool new additions. Let’s jump into the action and see what’s interesting.

LOGICAL COMBINATORS::MATCHES, :NOT

:MATCHES

This selector allows us can group and match items in our CSS document. Why is it so useful? see with your eyes...…

Instant of

ul.menu li a:link, ul.menu li a:hover, ul.menu li a:visited, ul.menu li a:focus { color: red; }

We can

ul.menu li a:matches(:link, :hover, :visited, :focus) { color: red; }

Simple, right? ;)

:NOT

:not, the simple negation pseudo-class, which now can take a list of selectors as parameters:…

p:not(.active, .visible) { color: red; }

The above code will apply COLOR: RED to ALL paragraphs that are not assigned active, visible class.

LOCATION PSEUDO-CLASSES::ANY-LINK, :LOCAL-LINK

:ANY-LINK

we will have more control over the styling of links…

Instant of

a:link, a:visited { color: red; }

We can

a:any-link { color: red; }

Simple, right? ;)

:LOCAL-LINK

You could give a different style to the links that target your home page, leaving all others elements:

nav :local-link { text-decoration: none; }
